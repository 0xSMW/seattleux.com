# Design critique format + rubric

## Format (30 minutes)

1. Context + goals (2 min)
2. Walkthrough (5 min)
3. Questions (5 min)
4. Feedback by rubric (15 min)
5. Decision + next steps (3 min)

## Rubric (score 1â€“5)

- Problem clarity
- Target user clarity
- IA / findability
- Interaction clarity
- Error handling
- Accessibility baseline
- Visual hierarchy
- Feasibility (tech/ops)

## Output

- Decision made
- Changes to make
- Open questions and owners
