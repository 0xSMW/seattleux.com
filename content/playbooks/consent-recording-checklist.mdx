# Consent and recording checklist

## Before the session

- [ ] Consent language prepared and accessible
- [ ] Recording tool tested (audio + screen)
- [ ] Backup plan if recording fails
- [ ] Note-taking template ready
- [ ] Participant has meeting link and time confirmed

## At the start

- [ ] Explain purpose and how data will be used
- [ ] Confirm consent to participate
- [ ] Confirm consent to record (if applicable)
- [ ] Confirm they can stop at any time

## After the session

- [ ] Store files in the agreed location
- [ ] Label recordings consistently
- [ ] Remove personal identifiers where appropriate
